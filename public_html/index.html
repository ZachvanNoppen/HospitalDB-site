Purpose

The purpose of the project is to provide us with a practical experience in designing and developing a medium scale database management application. A hospital operation would typically entail large amount of information and data pertaining to the hospital management and operation. This information includes patients records data, doctors and nursing staff operations data, hospital administration data, medical equipments inventories data, lab tests data, and medical supplies stocks information.

Our Plan

The scope of the project is limited to building the database structure consisting mainly of the database tables, indexes, and constraints using the DDL language. A sample hospital operation data will be populated into the database to help exercise and validate the database operations integrity and correctness using the SQL language. The project will not include building a frontend user interface. The data manipulation of the database content will be limited to using the command-line SQL language.
